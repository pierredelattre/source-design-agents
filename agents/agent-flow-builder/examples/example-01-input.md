# Input: User stories — Login and onboarding

**US-01:** As a new user, I want to create an account with my email so that I can access the app.

**US-02:** As a new user, I want to verify my email address so that my account is secure.

**US-03:** As a new user, I want to complete a short onboarding so that I understand the key features.

**US-04:** As a returning user, I want to log in with my email and password so that I can access my account.

**US-05:** As a returning user, I want to reset my password if I forget it so that I can regain access.
